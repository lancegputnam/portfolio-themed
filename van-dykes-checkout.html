Van Dykes Restorers Mobile Checkout

Van Dykeâ€™s Restorers was in need of a more user-friendly mobile checkout process. The former checkout was one lengthy checkout with a crowded interface of many small input fields and an abundance of text. I was tasked with designing the wireframes, building out the application with HTML and CSS, and working with a developer to implement features with Javascript. The end result was a clean and contemporary checkout process.

The checkout process was divided into four small pages, rather than one long one, complete with a progress bar at the top. This progress bar was put in place to let the users know exactly where they are and how much further they have to go. Real-time validation features were also added to the input fields so users are given confirmation with each input field.

The layout was designed with a simple color palette of greys, midnight blues, and fern greens. The flat buttons were also styled to take up more screen real estate. The new design and features of the mobile checkout process created a simple and enhanced experience to speed up the process and decrease abandoned carts.
